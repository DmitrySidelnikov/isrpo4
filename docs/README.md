# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ah/2

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = abc
## Area functions
### Circle: S = πR²
На вход получаем число r, возвращает r² * π

    Пример:
    Ввод:
    r=2
    Вывод:
    area=12.56 

### Rectangle: S = ab
На вход получаем числа a и b, возвращает a * b

    Пример:
    Ввод:
    a=3
    b=4
    Вывод:
    area=12.00

### Square: S = a²
На вход получаем число a, возвращает a²

    Пример:
    Ввод:
    a=5
    Вывод:
    area=25.00

### Triangle: S = ah/2
На вход получаем числа a и h, возвращает (a * h) / 2

    Пример:
    Ввод:
    a=6
    h=4
    Вывод:
    area=12.00

## Perimeter functions
### Circle: P = 2πR
На вход получаем число r, возвращает 2 * π * r

    Пример:
    Ввод:
    r=3
    Вывод:
    perimeter=18.84

### Rectangle: P = 2a + 2b
На вход получаем числа a и b, возвращает 2*(a + b)

    Пример:
    Ввод:
    a=5
    b=7
    Вывод:
    perimeter=24.00

### Square: P = 4a
На вход получаем число a, возвращает 4 * a

    Пример:
    Ввод:
    a=6
    Вывод:
    perimeter=24.00

### Triangle: P = a + b + c
На вход получаем числа a, b и c, возвращает a + b + c.

    Пример:
    Ввод:
    a=3
    b=4
    c=5
    Вывод:
    perimeter=12.00
